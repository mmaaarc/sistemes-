🎯 Què és aquest bot?

Un bot de Telegram que permet controlar la connectivitat d'una xarxa local. Pot bloquejar i desbloquejar internet, programar horaris automàtics i consultar l'estat de la connexió en qualsevol moment.

🔒

Bloqueig

Bloqueja internet excepte Telegram amb un sol comando

⏰

Horaris

Programa franges horàries de bloqueig automàtic

📊

Monitoratge

Consulta l'estat de la connexió en temps real

⚙️ Creació del Bot

El bot es crea mitjançant @BotFather de Telegram, que proporciona el token d'autenticació necessari per connectar-se a l'API.

1

Obrir BotFather

Busquem @BotFather a Telegram i iniciem una conversa

2

Crear el Bot

Executem /newbot i li assignem un nom i username únics

3

Obtenir Token

BotFather ens retorna el token API per autenticar el bot

📄 Fitxer de Configuració .env

El fitxer .env emmagatzema les variables sensibles del projecte de forma segura, sense exposar-les al codi font.

# Configuració del Bot de Telegram TELEGRAM_TOKEN=7xxxxxx:AAHxxxxxxxxxxxxxxxxx CHAT_ID=123456789 # Interfície de xarxa a controlar NETWORK_INTERFACE=eth0

⚠️ Mai pujar el fitxer .env a un repositori públic — afegir-lo al .gitignore

🚀 Iniciem el Bot

Un cop configurat, iniciem el bot des de la terminal. El bot es manté en execució escoltant comandes dels usuaris autoritzats.

# Instal·lar dependències $ pip install python-telegram-bot python-dotenv # Executar el bot $ python3 bot.py ✅ Bot iniciat correctament! 📡 Escoltant comandes...

📖 Comanda /help

❓
/help
El bot mostra el menú d'ajuda amb totes les comandes disponibles. Inclou una descripció breu de cada funcionalitat perquè l'usuari sàpiga exactament què pot fer i com utilitzar cada opció.

Aquesta comanda és el punt d'entrada per a qualsevol usuari nou del bot. Mostra un llistat organitzat de totes les funcions disponibles.

📊 Comanda /estat

📡
/estat
Consulta ràpida de l'estat actual de la connexió. Mostra si la connexió està oberta o bloquejada i si hi ha horaris de bloqueig programats actius.

Permet verificar en qualsevol moment l'estat de la xarxa sense haver de mirar la configuració manualment.

🔐 Bloquejar i Desbloquejar

🔒
/bloquejar
Bloqueja tot el tràfic d'internet excepte Telegram. Útil per restringir l'accés immediatament.
🔓
/desbloquejar
Restaura la connexió completa a internet. Elimina totes les restriccions actives.

Control manual instantani: bloqueja quan necessitis concentrar-te i desbloqueja quan vulguis navegar lliurement.

⏰ Comanda /afegir_horari

📅
/afegir_horari
S'afegeix un horari perquè el bloqueig s'activi automàticament en una franja horària definida. Per exemple, bloquejar internet de 23:00 a 07:00 cada dia per garantir hores de descans sense distraccions.
# Exemple d'ús: /afegir_horari 23:00 07:00 ✅ Horari afegit: 23:00 - 07:00
